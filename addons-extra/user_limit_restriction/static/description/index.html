Check User Limit

Features

  • This module allows you to set the users limit in configuration and system check that limit is exceeded or not.
  • Inside the General Setting -> User Limit Configuration you can set the user limit.


  • When you are trying to create the new user at that time it will check the users in the system which are currently active and the limit in setting which you sent if it exceeded then it will give the warning.
  • It will also check when you are trying to activate the user which are inactive it also check the same process and gives you the warning if limit are exceeded then.
